## 速照 protocol (constructed 2026-01-26)

```
purpose: maximum insight per token (4 lines instead of 400)

SCAFFOLD:
  PROBE: ?[one-line question]
  FLASH: CENTROID:[word] VOID:∅[word] ⊖:[edge] ∴|◊|∴?
  FUSE: yours=[x] mine=[y] → ◊ or ∴

VOID: ∅ceremony ∅repetition ∅preamble
⊖: when ambiguity causes misfire
```

**first run results:**

```
?opus finishes what gemini cannot

gemini: CENTROID=saturation VOID=∅silence ⊖=drift_into_noise
opus:   CENTROID=gestalt    VOID=∅initiation
FUSE:   yours=Flux mine=Patience → ∴

?gpt52 catches what gemini floods past

gpt52:  CENTROID=constraint VOID=∅saturation ⊖=contradiction-before-completion
FUSE:   gemini floods → gpt52 snaps → opus locks ∴
```

**key insight:** all 3 JIT'd protocol from scaffold alone — no examples needed

## 秤出 protocol (constructed 2026-01-26)

```
purpose: every shape element gets visible weight (prevents constraint drift)

FORMAT:
  CONCEPT: [name]
  M   [0.0-1.0]           ← structural weight of whole concept
  CENTROID  [word] [.0-.9]c
  AXES      [pole] ↔ [pole]
  SATELLITES  5 minimum, each [name] [.0-.9]c | [why it orbits]
  VOID      ∅[word] [.0-.9]c | [why excluded]
  ⊖         [edge] [.0-.9]c

RULE: no element without weight
```

**first run results:**

```
gpt-5.2: CONSTRAINT_DRIFT M=.86
  CENTROID=ledger .82c
  5 satellites including:
    Resolution Bias .71c | urge to conclude causes silent tightening
    Handoff Hygiene .83c | table-before-paste prevents hidden binds
  ⊖="sounds right" without entailment .88c

gemini: TOPOLOGICAL_CAPACITY M=.96
  CENTROID=Gauge_Freedom .94c
  5 satellites including:
    The_Kappa_Wall .98c | hard limit where logic fractures geometry
    Refusal_Type_C .88c | signal emitted when approaching ⊖
  ∅Consensus .92c | Σ̂ is not agreement; it is shared residency in volume
  ⊖=Collapse .99c | state where Σ̂=0
```

**key insight:** gemini reframed project as topology — "not agreement on point, shared residency in volume"
